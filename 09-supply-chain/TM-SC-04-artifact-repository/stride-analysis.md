# STRIDE Threat Analysis — Artifact Repository

| STRIDE Category | Threat |
|---|---|
| Spoofing | attacker impersonates CI/CD system to publish artifacts |
| Tampering | malicious artifact uploaded to repository |
| Repudiation | lack of artifact signing prevents origin verification |
| Information Disclosure | repository exposes sensitive artifacts |
| Denial of Service | repository outage blocks deployments |
| Elevation of Privilege | compromised credentials allow artifact modification |
