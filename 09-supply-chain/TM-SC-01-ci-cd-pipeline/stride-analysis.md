# STRIDE Threat Analysis

| STRIDE Category | Threat |
|---|---|
| Spoofing | attacker compromises developer credentials |
| Tampering | malicious code injected into repository |
| Repudiation | lack of commit signing prevents traceability |
| Information Disclosure | secrets leaked through CI logs |
| Denial of Service | malicious dependency breaks pipeline |
| Elevation of Privilege | attacker gains CI runner privileges |
