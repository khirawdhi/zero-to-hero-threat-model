# STRIDE Threat Analysis — Deployment Pipeline

| STRIDE Category | Threat |
|---|---|
| Spoofing | attacker impersonates deployment pipeline or service account |
| Tampering | deployment configuration modified to deploy malicious artifact |
| Repudiation | lack of deployment audit logs prevents traceability |
| Information Disclosure | secrets exposed during deployment |
| Denial of Service | malicious deployment disrupts runtime environment |
| Elevation of Privilege | compromised pipeline gains privileged infrastructure access |
