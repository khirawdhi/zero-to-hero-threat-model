# SolarWinds Supply Chain Attack

The SolarWinds attack was one of the most significant software supply chain compromises ever discovered.

Attackers infiltrated the build environment of the SolarWinds Orion platform and inserted malicious code into software updates distributed to thousands of organizations.

These updates were digitally signed and trusted by downstream systems.

## Impact

- thousands of organizations affected
- compromise of government and enterprise networks
- long-term espionage campaigns

## Attack Category

Software Supply Chain Attack
