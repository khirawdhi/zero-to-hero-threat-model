# How to Use This Repo

## For learners
1. Pick one reference architecture in `/02-reference-architectures/`
2. Draw a DFD with trust boundaries
3. Run STRIDE per data flow + per boundary
4. Write mitigations using the Controls Library
5. Create a small test plan to validate controls

## For teams
- Standardize on the templates in `/01-templates/`
- Use one threat model folder as a baseline per system pattern
- Track findings in `risk-register.csv` and close them like engineering work
